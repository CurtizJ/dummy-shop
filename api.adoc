[[_item_post]]
=== Add item
....

....


==== Description
Add single item


==== Parameters

[options="header", cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|**Body**|**item** +
__required__|Item to add|<<_items_item,items.Item>>
|===


==== Responses

[options="header", cols=".^2a,.^4a"]
|===
|HTTP Code|Schema
|**200**|No Content
|===

* `application/json`

* items

....
